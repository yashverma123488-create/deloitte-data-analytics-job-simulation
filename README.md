## Task 1: Factory Telemetry Analysis

### Problem Statement

Daikibo, a global manufacturing company, collected telemetry data from machines across four factories:

- Daikibo Factory Meiyo (Tokyo, Japan)
- Daikibo Factory Seiko (Osaka, Japan)
- Daikibo Berlin (Berlin, Germany)
- Daikibo Shenzhen (Shenzhen, China)

The objective was to analyze machine telemetry data and answer the following business questions:

1. Which factory experienced the highest number of machine breakdowns?
2. Which machine type broke down most frequently in that factory?

### Approach

- Processed and analyzed machine telemetry data.
- Built an interactive Tableau dashboard.
- Compared machine health across factories and device types.

### Key Findings

- Daikibo Factory Seiko recorded the highest number of unhealthy machines.
- Laser Welder was the machine type with the highest failure count.

## Task 2: Gender Pay Equality Investigation

### Problem Statement

Daikibo Industrials received internal complaints regarding potential gender pay inequality across different job roles and factory locations.

The Forensic Technology team generated an Equality Score for each job role and location, where:

- Scores range from -100 to +100
- 0 represents perfect equality

The objective was to classify each Equality Score into meaningful categories to identify potential pay inequality concerns.

### Classification Rules

| Equality Score | Classification |
|---------------|---------------|
| -10 to +10 | Fair |
| Less than -10 or Greater than +10 | Unfair |
| Less than -20 or Greater than +20 | Highly Discriminative |

### Approach

- Created an additional "Equality Class" column in Excel.
- Applied classification logic using Excel formulas.
- Categorized all records into Fair, Unfair, and Highly Discriminative groups.

### Outcome

Successfully automated the classification process and identified areas requiring further investigation for potential pay inequality.
